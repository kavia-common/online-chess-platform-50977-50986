# Chess Frontend

This React app implements a local two-player chess game with:
- Full rules: legal moves, checks, checkmate, stalemate, castling, en passant, promotions.
- Click-to-move interactions, highlights for selected square and legal targets.
- Move history and controls (Undo, Reset/New Game).
- Light theme with #3b82f6 primary and #06b6d4 accents; responsive board.

Run:
- npm start (port 3000)
- npm test (basic engine tests + smoke test)
